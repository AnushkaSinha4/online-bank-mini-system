# online-bank-mini-system
An Online Bank Mini System is built using HTML, CSS and JavaScript which allow users to create accounts and perform secure banking transactions with proper validation.
This project is developed as part of and assignment to demonstarte validation logic, transaction handling and UI design.
## Live Demo
Live Deployment URL:

GitHub Repository:  
https://github.com/AnushkaSinha4/online-bank-mini-system
##  Features
Create new bank account  
Deposit money  
Withdraw money  
Transfer money between accounts  
KYC verification validation before transfer  
Balance validation for withdrawals and transfers  
Error handling with user-friendly messages  
Clean and responsive UI  

## Data Model
Each account contains:
Account Number
Holder Name
Balance
KYC Verification Status (true / false)

## Functional Validations
Account number must be unique during creation  
Account must exist for deposit and withdrawal  
Withdrawal only allowed if sufficient balance  
Transfer allowed only if:
Sender KYC is verified
Sender has sufficient balance  
Proper error messages for invalid operations  

## Tech Stack
HTML  
CSS  
JavaScript  
GitHub Pages (Deployment)

## How to Run Locally

1. Download or clone this repository  
2. Open `index.html` in any web browser  
3. Start creating accounts and performing transactions  

## UI Preview
The application provides a clean and responsive card-based interface with an output message panel for transaction results.
